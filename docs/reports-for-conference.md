Отчет по мероприятиям (для интерфейса и скачиваемый)
Обновлено 16 октября 2025, 13:32

Данный отчет предназначен для отслеживания результатов конференции и других мероприятий. Результат - это изменения продаж брендов проекта EVERYCAR.

Отчет по мероприятиям формируется на основе загруженных в систему Логин2. После формирования отчета - он продолжает существовать и обновляться пока администратор не удалит его.

Администратор загружает Логин 2 (списком, может быть несколько Логин 2) и дает имя отчету.

Отчет содержит несколько закладок с разной степенью детализации. Начало периода 1 января 2024 года. Перед каждым графиком стоит две цифры - кол-во Логин 2, которые загружены, кол-во Логин 2 по которым выведена информация в график.

1. Общий (по всем брендам проекта) - столбчатая диаграмма или график с выбором брендов и периода (период - с месяц по месяц). В график выводится информация за 2024 и 2025 год в выбранном периоде.

1.1. Закупка в руб.

1.1.1. По умолчанию показывает весь объем закупок данных СТО в руб.

1.1.2. Бренды проекта - объем закупок, выбранных СТО в руб.

1.1.3. Выбор отдельного бренда из списка брендов проекта - объем закупок выбранных СТО в руб.

1.2. Закупка в шт.

1.2.1. По умолчанию показывает весь объем закупок данных СТО в шт.

1.2.2. Бренды проекта - объем закупок, выбранных СТО в шт.

1.2.3. Выбор отдельного бренда из списка брендов проекта - объем закупок выбранных СТО в шт.

1.3. Количество СТО с закупкой в шт.

1.3.1. По умолчанию показывает кол-во СТО в шт., которые имеют закупки, которые не равны 0

1.3.2. Кол-во СТО в шт., которые имеют закупки брендов проекта, которые не равны 0

1.3.3. Кол-во СТО в шт., которые имеют закупки выбранного бренда проекта, которые не равны 0

1.4. Средняя закупка на 1 СТО в руб.

1.3.1. По умолчанию показывает среднюю закупку в руб. на 1 СТО (формула - пункт 1.1.1. / пункт 1.3.1)

1.3.2. Средняя закупка в руб. брендов проекта на 1 СТО (формула - пункт 1.1.2. / пункт 1.3.2)

1.3.3. Средняя закупка в руб. выбранного бренда проекта на 1 СТО (формула - пункт 1.1.3. / пункт 1.3.3)

1.5. Средняя закупка на 1 СТО в шт.

1.3.1. По умолчанию показывает среднюю закупку в шт. на 1 СТО (формула - пункт 1.2.1. / пункт 1.3.1)

1.3.2. Средняя закупка в шт. брендов проекта на 1 СТО (формула - пункт 1.2.2. / пункт 1.3.2)

1.3.3. Средняя закупка в шт. выбранного бренда проекта на 1 СТО (формула - пункт 1.2.3. / пункт 1.3.3)
